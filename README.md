This is the browser version of the game, optimized for fast loading (52 MB).

### 🎮 [PLAY NOW] : (https://pig-and-pancakes-team.github.io/Pig-and-pancakes-web-version/Web3/)

---

### ⚠️ Important for players:
* **Fullscreen:** If the window is small, simply move the browser tab into a separate window – the scale will adjust automatically.
* **Bugs:** In the web version, sprites may flicker or disappear,  the game cannot be completed (only the menu and the open world itself are available), the second player and all NPCs except for the green pancake enemies do nothing – this is due to my export and the "features" of the browser engine.
* **Sound:** I cut the sound in the web version to reduce the size.

### ⚠️ Web-Camera Warning: Из-за конфликта браузерных скриптов при возврате в меню через SPACE камера может «сжаться».
Решение: Для идеального отображения после завершения сессии просто обновите страницу (F5). Это гарантирует сброс всех буферов и корректный масштаб!
